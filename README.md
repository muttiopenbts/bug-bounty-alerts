# Bug Bounty Alerts
HackerOne to Pager Duty integration for automatic alerts.

# Install
* Clone report
```
git clone <THIS REPO>
```
* Install packages
```
cd <REPO NAME>
pipenv install
```
# Configure
```
export PD_SERVICE_ID=<>
export PD_TOKEN=<>
export H1_PROGRAM_NAME=<>
export H1_API_KEY_NAME=<>
```
# Run
```
python ./bug_bounty_alart.py
```
